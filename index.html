Auditoría de Identidades con IA — Producción
Tenant real · 300 usuarios · Microsoft Graph API + Claude Code + Python
Producción Real Claude Code
Microsoft Graph APIClaude Code CLIPython 3.xOAuth 2.0Entra ID ProtectionMSAL

Construido en producción usando Claude Code directamente en el terminal. Creé una App Registration en Azure con OAuth 2.0 (Client Credentials), configuré los permisos necesarios en Graph API y desarrollé dos scripts Python que extraen todos los usuarios del tenant, analizan sus señales de riesgo y generan un informe priorizado con scoring 0-10. El sistema detectó 12 identidades en riesgo en menos de 2 minutos — incluyendo usuarios con credenciales comprometidas, accesos legacy sin MFA y logins desde países no habituales.
